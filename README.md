# Calculator with TypeScript

Math calculator with basic arithmetic operations without eval. It based on reverse polish notation.

```npm run calc```

Then you can calc with complex math operations.

Examples:

```
> 10 + 10
Result: 20
> 10 + 10 * 20 - 30
Result: 180
> 19 + -10
Result: 9
> 2**
Result: 4
> 3 ** 3
Result: 27
> sin(3.14)
Result: 0
```
